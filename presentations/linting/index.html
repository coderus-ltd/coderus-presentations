Linting

What it is, and a concrete example.

What is it?

The process of running a static analysis tool to detect problems with your code

A Quick History

  • Name comes from "lint" a Unix tool for detecting bugs in C code
  • First seen in V7 of Unix in 1979
  • "Linter" became a name for any tool that runs over code to check for potential issues

What can linters do?

  • Look for coding errors
  • Highlight optimizations
  • Enforce code style

A Concrete Example

SwiftLint

  • A linter for Swift
  • Developed by Realm
  • Integrates with Xcode via a build script

SwiftLint

SwiftLint - Custom Rules

custom_rules:
  pirates_beat_ninjas:
    included: ".*.swift"
    name: "Pirates Beat Ninjas"
    regex: "([n,N]inja)"
    match_kinds:
      - comment
      - identifier
    message: "Pirates are better than ninjas."
    severity: error
  no_hiding_in_strings:
    regex: "([n,N]inja)"
    match_kinds: string
